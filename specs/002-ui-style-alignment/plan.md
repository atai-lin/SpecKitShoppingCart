# Implementation Plan: UI Style Alignment (Atelier Design)

**Branch**: `002-ui-style-alignment` | **Date**: 2026-04-11 | **Spec**: [specs/002-ui-style-alignment/spec.md](spec.md)
**Input**: Feature specification from `/specs/002-ui-style-alignment/spec.md`

## Summary

The goal of this feature is to align the SpecKit Shopping Cart's UI with the "Atelier" design provided in Figma. This involves a complete visual overhaul of the product listing page, including a new color palette, typography (Manrope & Work Sans), an asymmetrical product grid, and enhanced filtering options (Material & Color). The implementation will use Tailwind CSS to ensure high-fidelity styling and maintainable design tokens.

## Technical Context

**Language/Version**: ASP.NET Core 10
**Primary Dependencies**: EF Core, Swagger, ILogger, bcrypt, JWT, Tailwind CSS
**Storage**: SQLite (via EF Core Code First)
**Testing**: xUnit with 80% coverage target
**Target Platform**: Web (Responsive)
**Project Type**: ASP.NET Core Razor Pages
**Performance Goals**: Page Load < 1s, API < 200ms
**Constraints**: Mandatory HTTPS, Shared Layout, 60min JWT
**Scale/Scope**: Focus on `Index.cshtml` (Product Listing), `_Layout.cshtml` (Header/Footer), and Product data model extension.
